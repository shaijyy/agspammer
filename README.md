# agspammer
AnswerGarden Spammer

Uses FreeSimpleGUI and requests.  
Spams An Answer(s) in an AnswerGarden board.  
Funny to do in class XD  

requirements:  
FreeSimpleGUI  
requests  

USAGE:  
Just run the python file, make sure to install all dependencies!  
```
python main.py
```  
install dependencies:  
```
pip install -r requirements.txt
```  
or in a uv-managed python project:  
```
uv add -r requirements.txt
```
